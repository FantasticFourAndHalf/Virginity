# Virginity
Restaurant Web Application
 **Ресторан _" Virginity :cherry_blossom:"_**    
 У даному веб-додатку пропонуються такі послуги нашого ресторану:
 - Замовлення столика;
 - Складання меню;
 - Доставка страв;
 
 1. Замовлення:
    - страви
      - Назва страви
      - Інгрідієнти
      - Ціна
      - Час приготування
      - Білки
      - Жири
      - Вуглеводи
    - столики
      - Клієнт
      - № столика
      - Меню
      - Дата і час резерву
